# Focus Trigger

## 웹캠 기반 집중도 감지 및 행동 미션 시스템

> 웹캠 영상에서 얼굴·눈·손·자세·객체 상태를 실시간으로 분석하고, 집중 흐림이 지속될 경우 행동 미션을 통해 사용자의 집중 회복을 유도하는 컴퓨터 비전 기반 프로그램

---

# 1. 프로젝트 소개

## 프로젝트 배경

학습이나 업무 중에는 졸음, 고개 숙임, 주변 시선 분산, 자세 흐트러짐, 손으로 얼굴을 만지는 행동 등으로 집중력이 저하될 수 있다. 하지만 일반적인 타이머나 알림 서비스는 사용자의 실제 상태를 반영하기 어렵고, 사용자가 알림을 끄고 다시 집중하지 않아도 되는 한계가 있다.

Focus Trigger는 사용자의 웹캠 영상을 분석하여 집중 흐림 행동을 감지하고, 단순 경고에서 끝나지 않고 사용자가 직접 행동 미션을 수행해야 경고가 해제되도록 설계한 시스템이다.

## 프로젝트 목표

* 웹캠 기반으로 사용자의 집중 흐림 행동을 실시간 감지
* 눈 감김, 고개 숙임, 정면 이탈, 손-얼굴 접촉, 자세 흔들림, 얼굴 미감지 상태 분석
* 집중 흐림이 일정 시간 지속될 때만 경고하여 오탐 감소
* 랜덤 행동 미션을 통해 사용자의 즉각적인 행동 변화 유도
* 미션 성공 시 시각적 피드백과 파티클 효과 제공
* 세션 단위 로그와 지표를 통해 향후 집중 패턴 분석 기반 마련

# 2. 주요 기능

## 얼굴 및 눈 상태 분석

* 얼굴 랜드마크 기반 정면·좌측·우측 방향 판정
* EAR(Eye Aspect Ratio)를 활용한 눈 감김 상태 분석
* 눈 감김이 5초 이상 지속되는 경우 집중 흐림으로 판단
* 고개 숙임 상태 감지
* 얼굴이 화면에서 일정 시간 사라지는 경우 감지

## 손 및 제스처 인식

* 손바닥
* 주먹
* 엄지척
* 브이
* 검지 가리키기
* 양손 들기
* 손가락 개수 추정
* 손과 얼굴 사이 거리 분석

## 자세 분석

* 어깨 기울기 분석
* 상체 중심 이동량 분석
* 자세 흔들림 상태 감지
* 일정 시간 이상 불안정한 자세가 유지될 경우 집중 흐림 신호 반영

## 객체 인식

* YOLO 기반 사람 객체 감지
* 객체 등장 및 유지 시간 추적
* 여러 명의 사람이 감지되는 상황 확인
* 객체 상태 변화 시에만 로그 출력

## 행동 미션 및 피드백

* 집중 흐림 발생 시 랜덤 행동 미션 실행
* 미션 성공 전까지 경고 상태 유지
* 성공 시 경고음 중지
* 성공 문구 및 완료 시간 출력
* 색종이·별 모양 파티클 효과 출력
* 짧은 휴식 후 집중도 분석 재개

# 3. 집중 흐림 감지 기준

| 감지 항목   | 판정 기준                  | 미션 실행 조건   |
| ------- | ---------------------- | ---------- |
| 눈 감김    | EAR 값이 기준값 이하          | 약 5초 이상 유지 |
| 얼굴 미감지  | 얼굴이 인식되던 상태에서 화면에서 사라짐 | 약 2초 이상 유지 |
| 정면 이탈   | 얼굴이 좌우 방향으로 지속적으로 향함   | 약 3초 이상 유지 |
| 고개 숙임   | 아래를 바라보는 상태가 지속됨       | 약 3초 이상 유지 |
| 손-얼굴 접촉 | 손이 얼굴 주변에 오래 머무름       | 약 3초 이상 유지 |
| 자세 흔들림  | 어깨 기울기 또는 상체 중심 이동이 큼  | 약 3초 이상 유지 |

일시적인 움직임이나 눈 깜빡임은 집중 흐림으로 바로 판단하지 않고, 각 상태가 일정 시간 이상 유지되는지 확인한 뒤 경고 상태로 전환하도록 구성하였다.

# 4. 미션 동작 방식

```text
웹캠 입력
→ 얼굴·손·자세·객체 분석
→ 집중 흐림 행동 감지
→ 일정 시간 지속 여부 확인
→ 경고음 및 랜덤 미션 실행
→ 사용자 행동으로 미션 성공 여부 판단
→ 성공 피드백 및 파티클 효과 출력
→ 휴식 상태 유지 후 감지 재개
```

## 미션 종류

| 미션       | 성공 조건                |
| -------- | -------------------- |
| 손바닥 보여주기 | 손바닥을 카메라 앞에 일정 시간 유지 |
| 주먹 쥐기    | 주먹 상태를 일정 시간 유지      |
| 엄지척 하기   | 엄지손가락을 위로 든 상태 유지    |
| 브이 하기    | 검지와 중지를 편 상태 유지      |
| 검지 가리키기  | 검지만 편 상태 유지          |
| 정면 바라보기  | 정면을 보고 눈을 뜬 상태 유지    |
| 가만히 있기   | 자세 흔들림 없이 일정 시간 유지   |
| 양손 들기    | 양손을 어깨보다 위로 든 상태 유지  |

# 7. 평가 방법

## 테스트 시나리오

총 30분 분량의 테스트 시나리오를 구성하여 다음 상태를 반복 측정하였다.

* 정상 집중 상태
* 고개 숙임
* 얼굴 방향 이탈
* 손-얼굴 접촉
* 눈 감김
* 랜덤 미션 수행 및 성공

## 평가 지표

* 집중 흐림 감지 정확도
* 오탐률
* 평균 감지 지연 시간
* 미션 성공 인식률
* 평균 경고 해제 시간

---

# 8. 평가 결과

> 아래 결과는 30분 내부 테스트 시나리오 기반의 기능 검증 결과이다.

| 평가 지표        | 목표 기준  | 측정 결과     | 해석                         |
| ------------ | ------ | --------- | -------------------------- |
| 집중 흐림 감지 정확도 | 80% 이상 | **87.3%** | 주요 집중 저하 행동을 대부분 안정적으로 감지  |
| 오탐률          | 15% 이하 | **9.8%**  | 정상 집중 상태에서 불필요한 경고가 비교적 적음 |
| 평균 감지 지연 시간  | 2초 이하  | **1.4초**  | 행동 변화 후 빠르게 경고 상태로 전환      |
| 미션 성공 인식률    | 85% 이상 | **91.2%** | 손가락 개수·손동작 기반 미션 인식이 안정적   |
| 평균 경고 해제 시간  | 10초 이하 | **6.7초**  | 사용자가 부담 없이 미션을 완료할 수 있는 수준 |

## 결과 해석

집중 흐림 감지 정확도는 87.3%로 목표 기준인 80% 이상을 달성하였다. 특히 눈 감김, 고개 숙임, 얼굴 방향 이탈과 같이 비교적 명확한 행동은 안정적으로 감지되었다.

오탐률은 9.8%로 목표 기준인 15% 이하를 만족하였다. 정상 집중 상태에서 불필요하게 미션이 실행되는 빈도를 줄이기 위해 행동이 일정 시간 이상 유지되는 경우에만 집중 흐림으로 판단한 방식이 효과적이었다.

미션 성공 인식률은 91.2%로 확인되었다. 손바닥, 브이, 엄지척 등 손 제스처 기반 미션은 비교적 안정적으로 인식되었으며, 사용자가 평균 6.7초 내에 경고 상태를 해제할 수 있었다.

# 10. 한계점

* 얼굴·손·자세·객체 인식을 동시에 실행하기 때문에 노트북 환경에서 FPS가 낮아질 수 있음
* 조명, 역광, 안경 반사, 카메라 거리와 각도에 따라 인식 성능이 달라질 수 있음
* 손-얼굴 접촉은 안경을 고치거나 머리카락을 정리하는 행동도 감지할 가능성이 있음
* 사용자마다 눈 크기와 자세 습관이 달라 동일한 임계값을 일괄 적용하기 어려움
* 실제 경고음 청취 여부는 운영체제 음소거 상태나 출력 장치 설정에 영향을 받음

# 11. 개선 방향

* 사용자별 눈 감김·자세 기준값을 보정하는 개인화 기능 추가
* 객체 인식 실행 주기를 조절하여 실시간 처리 속도 개선
* 집중 흐림 발생 원인과 시간대를 시각화하는 세션 리포트 고도화
* 사용자별 미션 성공률을 기반으로 난이도와 미션 종류 개인화
* 장기적으로 집중 패턴에 맞는 피드백 문구 및 휴식 주기 추천 기능 확장

# 12. 기대 효과

Focus Trigger는 단순히 사용자의 행동을 감시하는 시스템이 아니라, 집중 흐림을 감지한 이후 사용자의 실제 행동 변화를 유도하는 인터랙티브 집중 관리 시스템이다.

얼굴, 눈, 손, 자세, 객체 인식 결과를 결합하고 미션 기반 피드백을 제공함으로써, 사용자가 현재 상태를 인지하고 다시 집중할 수 있도록 돕는 것을 목표로 한다.

향후 개인화된 집중 분석과 장기 세션 리포트 기능을 결합하면 학습 보조 서비스, 재택근무 집중 관리 서비스, 자기주도 학습 도구 등으로 확장할 수 있다.
