# 화면·미션·경고음 보정본 적용 안내

이 ZIP은 **압축 안에 다시 폴더가 한 겹 생기지 않도록**, `main.py`가 바로 보이는 형태로 만들었습니다.

## 왜 미션이 없었나

정상 화면에서는 미션이 항상 보이지 않습니다.

- 눈 감김: 5초 이상 유지
- 얼굴 정면 이탈: 3초 이상 유지
- 고개 숙임: 3초 이상 유지
- 얼굴 미감지: 2초 이상 유지
- 손-얼굴 접촉 / 자세 흔들림: 3초 이상 유지

위 조건이 충족된 뒤에만 경고음과 랜덤 미션이 표시됩니다. 따라서 화면에 `Focus 0 (normal)`이라고 보일 때 미션 창이 없는 것은 정상입니다.

이번 보정본에는 미션·사운드 확인용 단축키를 추가했습니다.

| 키 | 기능 |
|---|---|
| `M` | 집중 흐림을 기다리지 않고 랜덤 미션을 즉시 시작합니다. 경고음도 같이 시작됩니다. |
| `B` | 짧은 두 번의 Windows 경고음 테스트를 요청합니다. |
| `D` | 얼굴·손·자세의 상세 랜드마크 표시를 켜고 끕니다. 기본은 꺼져 있습니다. |
| `S` | 현재 화면을 저장합니다. |
| `Q` / `ESC` | 종료합니다. |

## 적용 방법

1. 실행 중인 창에서 `Q`를 눌러 종료합니다.
2. 현재 실제 코드 폴더를 엽니다. `main.py`와 `config.py`가 **바로 보이는 폴더**여야 합니다.
3. 이 ZIP의 내용을 그 폴더에 풀고, 덮어쓰기 메시지가 나오면 **모두 덮어쓰기**를 선택합니다.
4. 같은 폴더 PowerShell에서 실행합니다.

```powershell
python main.py
```

## 사운드 코드만 따로 점검하기

```powershell
python sound_test.py
```

`[코드 점검 통과]`가 출력되면 Windows 경고음 호출까지는 정상적으로 요청된 것입니다.

다만 PC가 음소거 상태이거나 출력 장치가 다른 곳으로 잡혀 있으면 실제로 들리지 않을 수 있습니다. 그 부분은 나중에 소리를 들을 수 있을 때 `B` 키로 최종 확인하면 됩니다.

## 바뀐 점

- 카메라에 1280×720 / MJPG를 우선 요청하도록 변경
- 감지는 적당한 크기의 분석 프레임으로 돌려 FPS를 방어
- 화면 확장 **후** 박스·텍스트·랜드마크를 그려서 박스와 글자가 흐릿해지는 문제 개선
- 랜드마크 점 478개를 기본 화면에서 숨겨서 얼굴이 지저분하게 보이는 문제 개선
- `M` 미션 테스트, `B` 사운드 테스트 추가
- 눈 감김 등 판정 대기 중일 때 `몇 초 / 몇 초`인지 화면 아래 표시
- 경고음 호출 실패를 더 이상 조용히 무시하지 않고 콘솔·상태 영역에 표시
